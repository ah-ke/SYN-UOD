# SYN-UOD: Semantic Image Synthesis for Unknown Object Detection

Official implementation of the paper:

**Semantic Image Synthesis for Unknown Object Detection in Autonomous Driving Scenes**

## Overview
SYN-UOD is a discrepancy-driven framework for detecting unknown objects in autonomous driving scenes. The method leverages semantic image synthesis and discrepancy-aware learning to identify anomalies without requiring real unknown samples during training.

This repository provides the PyTorch implementation of SYN-UOD, including training scripts, evaluation code, and pretrained models.
